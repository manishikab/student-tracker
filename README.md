# AI-Powered Student Life Dashboard

A full-stack web application that helps students manage tasks, track wellness, and receive personalized AI coaching.

## Project Overview

Balancing academics, wellness, and daily routines can be challenging for students. The AI-Powered Student Life Dashboard centralizes to-dos, sleep logs, exercise, and journal entries in a single calendar view. 
Interactive AI coaches provide personalized guidance, helping students reflect on habits, stay motivated, and make informed decisions.

## Features

- Track tasks, sleep, exercise, and wellness journal entries in one place
- Interactive calendar view for all student activities
- AI-powered coaches using NLP for real-time chat guidance and personalized tips
- Full-stack architecture: FastAPI backend, React frontend
- Data persistence with SQLite
- Responsive and user-friendly interface
